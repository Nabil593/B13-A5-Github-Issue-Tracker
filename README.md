1️⃣ What is the difference between var, let, and const?<br>
Answer: var is a function-scoped variable that can be redeclared and reassigned. Let is a block-scoped variable that can be reassigned but not redeclared in the same scope. Const is a block-scoped variable that cannot be reassigned or redeclared after initialization.

---

2️⃣ What is the spread operator (...)?<br>
Answer: The spread operator expands elements of an array or properties of an object into individual values.

---

3️⃣ What is the difference between map(), filter(), and forEach()?<br>
Answer: map() creates a new array by transforming each element. filter() creates a new array with elements that pass a condition. forEach() executes a function for each element but does not return a new array.

---

4️⃣ What is an arrow function?<br>
Answer: An arrow function is a shorter syntax for writing functions in javaScript using the => arrow notation.

---

5️⃣ What are template literals?<br>
Answer: Template literals are strings written using backticks ( ) that allow embedded expressions using ${} and multi-line strings.
